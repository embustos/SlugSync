# SlugSync

SlugSync is an event calendar web app starter built with Vite and React. This
initial dashboard uses mock data only and includes placeholders for future
Instagram and Discord event sources.

## Getting Started

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

Vite will print the local URL in your terminal, usually
`http://localhost:5173/`.

## Project Structure

```text
src/
  App.jsx
  data/
    mockEvents.js
  pages/
    Dashboard.jsx
  main.jsx
  index.css
```

## Current Scope

- Vite + React frontend setup
- Dashboard page with a navbar, welcome section, calendar preview, and upcoming
  events
- Placeholder cards for future Instagram and Discord event integrations
- Mock data only

No backend code, scraping logic, or source integrations have been added yet.
